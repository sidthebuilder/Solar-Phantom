# Project: Solar Phantom (Forever Drone)

This project aims to design a solar-powered unmanned aerial vehicle (UAV) capable of continuous multi-day flight.

## Built With
*   [AeroSandbox](https://github.com/peterdsharpe/AeroSandbox)

## How to Run
1.  Install dependencies: `pip install -r requirements.txt`
2.  Run the design script: `python main.py`
